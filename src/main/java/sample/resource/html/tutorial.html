声飞简明教程
声飞的主要功能
●制作语言田野调查表
  软件预置了《方言调查字表》《词汇调查表》《句子调查表》《话语调查表》
  用户也可以将自己的excel调查表导入声飞软件
●录音录像
  调查表的每个条目单独存储为wav文件
  导出条目的音频文件，以编号命名，以中文命名，以英文命名，或混合命名音频文件
  用户可以将已录制的wav文件导入声飞软件，逐条导入或批量导入
  查看语图
●转写标注
  录入字、词、句、话语的音标注音和其他注释
  不带任何自定义输入法和字库文件，直接使用操作系统的字库，保证字符兼容
●导出（输出）
  导出字表、词表、句表、话语表的excel表格(*.xls/xlsx)
  导出字、词、句、话语的网页(*.html) ，如有录音录像的话，导出多媒体网页
●基本分析
  根据字表或词表的国际音标记音，生成《声韵调表》《同音字汇》《音节词汇表》
  根据句子表的国际音标记音，提取《词汇表》
  根据话语的音标记音和中文转写标注，提取《句子表》《词汇表》
  将多个调查表的音标记音组成《单字对照表》《词汇对照表》《句子对照表》

新建调查表（*.snf）
文件 > 新建字表
文件 > 新建词表
文件 > 新建句表
文件 > 新建话语表
操作示例一：新建一个汉语方言调查字表。
——单击文件 > 新建字表 > 保存。
——双击条目筛选列的保留二字，可将不需要的字隐藏；
——单击字表左边三角箭头，按shift选中多个记录，按工具栏隐藏按钮，可将多个条目隐藏；
——可使用工具栏的按钮，对表格的条目进行保留、隐藏、删除、显示操作；
——鼠标右键点击表头的编码、分级，可使用在当前列查找子命令，查找条目；
——鼠标右键点击分级>>在当前列查找，输入下面的分级代码：
    GD 代表调查声调的例字；GS 代表声调例字；GY代表韵母调查例字。G2代表扩充记录的单字。
——输入GD，按确定，查出声调例字。
——鼠标左键单击表头分级，对声调例字进行排序。
——按工具栏导出按钮，将声调例字表导出为xls表或html网页。
导入调查表
文件>>导入自制excel表
文件>>导入语保excel表
操作示例1：导入一个用户制作的词汇调查表。
重要提示：用户制作的调查表应是excel表，表头必须有编码、词条、音标注音。编码、词条的记录不能留空，编码不能重复。
——单击文件>导入自制excel表>>导入excel词表，弹出提示信息；
——选中测试表中文件夹自制表的词表1.xls；按打开，即可成功导入，自动新建一个调查词表001.snf。
操作示例2：导入语保excel词汇表。
重要提示：语保excel词汇表有2个：一个是录音用的词汇表；另一个是上交的词汇模板表。
录音用的词汇表表头有3个字段：民语、调查条目、注例。将民语改为条目，将调查条目改为编码，将注例改为注释。然后用示例1的自制表命令导入。
词汇模板表，直接按下面操作导入：
——单击文件>导入语保excel表>>导入语保词汇模板；
——选中测试表中文件夹语保表的语保词表1.xls，按打开，即可成功导入，自动新建一个调查词表002.snf.
调查表录音
操作示例：对一个声飞词汇调查表进行录音。
——单击文件>打开表，在列表中选调查词表001.snf,按打开；
——单击选项>录音模式，进入录音页面；
——单击工具栏刷新按钮；
——默认为手动录音，即一条一条地录：点选一个词条，按录音键，听到嘟声就说话，按停止键。
——勾选自动录音，设置每条的录音时间（例如2秒）之后，按下录音键，2秒后自动录下一个条目。
——自动录音建议每录完10个词条以后，停止并刷新一次，便于软件保存和整理录音文件，避免录音条目过多而死机。
——停止录音以后，按工具栏刷新按钮；
——使用工具栏的导入录音、导出录音、删除录音，可以逐个或批量导入外部的音频或视频文件，或导出音频文件，或删除音频文件。
——使用表头名称的鼠标右键菜单，进行列操作；
——使用条目头的鼠标右键菜单，可进行条目操作；
——按praat图标按钮，可以查看语图。
音标注音和标注
——打开一个已经有录音的声飞词汇表调查词表001.snf;
——单击选项>转写模式；进入转写页面，并打开一个《字符影射表》面板，将它拖放到右边空框里；
——勾选自动播放、循环播放；
——单击第一个空格条，一边播放录音，一边录入音标；
——遇到国际音标符号，在字符影射表中查找，找到以后，按选择、复制，再粘贴到音标转写空格条里；
——音标输入完毕，取消自动播放、循环播放；
——单击文件>制表模式，回到调查表全表页面。
音标查错
——单击文件>打开表，在列表中选中词汇调查表001.snf,打开；
——按工具栏刷新按钮，刷新词汇表；
——单击数据>音标查错，弹出提示信息，音标注音的符号使用要求；按确定；
——系统开始查错，每遇到可能的错漏，即弹出错误提示；
——如果不是错误，按忽略本条；如果属于错误，按继续查错。请注意：查错必须一条一条仔细地辨认！用户应有耐心，这是必要。
——查错完毕，系统将所有错误列表；
——点击错误条，在调查表中定位错误词条，进行修改；
——再进行第二次查错，修改，直到没有错误。
整理声韵调和同音字汇词汇
——单击文件>打开表，打开一个已有音标注音的声飞调查表，例如：词汇调查表001.snf；
——单击数据>整理声韵调;系统建议先进行音标查错，音标无错误以后，再整理声韵调。
——如果已经查错，则在弹出的对话窗中按忽略全部，系统弹出声韵调列表；
——按预设顺序，初步调整声母、韵母的排序；
——按箭头按钮，对个别声母、韵母排序进行调整；
——输出设置，设置声韵调表的例词个数，排版方式，导出的文件类型；按确定，即可将声韵调表导出为xls表格或html网页文件。
整理同音字汇词汇：
对于汉语方言字表，将所有音节的同音字归类列表；
对于词汇表，将包含该音节的所有词归类列表。
提取句子词表
    对于一个已经有音标注音、普通话词对译的句子调查表，可以将全部句子的每个词提取出来，生成一个词汇表。这对于核对句子中的单词记音，十分有用。
——单击文件>导入自制excel表>导入excel词表；
——选中自测表中的文件夹自制表的句表01.xls，按打开，导入这个表；
——单击数据>提取句子词表，系统弹出提示：建议先进行音标查错；确认无错以后，再按提取句子词表命令；
——系统弹出词表的输出格式设置窗口：横排还是竖排，输出xls表，还是网页文件；
——系统进行单词分词查错。句子表要求每个词用空格分词，系统会计算句子音标注音、普通话词对译的空格个数，如果空格数目不相等，则说明分词不一致。系统会弹出错误列表窗。根据列表中的空格统计信息，进行修改。
——如果空格数目不相等，系统将无法生成词表。
多表对照
    如果有多个字表、词表、句子表已经有音标注音，系统可以根据用户的选择，生成单字对照表、词汇对照表、句子对照表。
——单击文件>导入自制excel表>导入excel词表，
——选中测试表中文件夹自制表的词表2.xls，按打开；
——再选中词表3.xls，按打开；
——关闭当前打开的调查表；
——单击数据>多表对照，
——在上面的选择一个基准表列表中，选中词汇调查表0001.snf；
——在下面的对照表中选择词汇调查表002.snf、词汇调查表003.snf；
——按确定，即可生成对照表；如果有重复条目，则弹出提示信息；可用键盘Delete键删除；
——单击文件>导出，可以导出对照表的excel表格，或网页文件。
提取话语文本句子
    话语录音如果有音标注音、普通话词对译、普通话逐句意译，系统可以提取其中的句子，生成句子表。系统根据话语文本的句子标点符号切分句子。音标注音、普通话词对译、普通话逐句意译的文本，标点符号数目应相等，如果不相等，提取的句子表则会错位。用户可以根据错位的句子表，看到文本何处出现了标点符号错漏。
——单击文件>新建话语表，新建一个声飞话语调查表001.snf；
——选中编码D02A006的话题条目；
——鼠标右键点击音标注音对应的空格，弹出导入文本命令；
——选中测试表>>自制表的话语音标.txt文本文件，按打开，导入话语音标文本；
——鼠标右键点击普通话词对译对应的空格，导入测试表中的话语普通话对译.txt；
——鼠标右键点击普通话意译对应的空格，导入测试表中的话语普通话意译.txt；
——刷新页面；
——选中刚才导入的文本格，移动菜单下面的大字显示框右边的按钮，将文本字号缩小；
——单击数据>提取句子，系统检查标点符号，弹出标点符号数目不一致的提示信息；
——按继续>横排>输出excel表；[有的电脑excel可能是老版本，请到处xls文件格式]；即可成功导出句子列表。
——再单击数据>提取句子，试输出网页，看看结果如何。
提取话语词表
    话语录音如果有音标注音、普通话词对译、普通话逐句意译，系统也可以提取其中的单词，生成词汇表。系统根据话语文本空格切分单词。音标注音、普通话词对译、普通话逐句意译的文本句子中，空格数目应相等，如果不相等，提取的词表则会错位。
——单击文件>打开表，选中一个已有的话语调查表001.snf；如果没有则新建。
——单击数据>提取词汇，
——按继续>输出网页，到处一个网页文件的词汇表；
——打开这个网页文件，查看词表；如果词汇表错位，则在话语调查表的大字显示框中修改。
组织多媒体网页语料库
    声飞可以将输出的各种网页文件，组织成一个多媒体的语料库或有声档案包，便于上传到网络服务器共享。
——单击语档>说话人信息表，填写发音人和说话人信息；
——单击语档>调查表元数据，填写每个调查表的元信息；
——单击语档>创建语档网页，按页面提示，逐步操作，最后输出语档。
音系表（声韵调表）应先用菜单命令数据>整理声韵调生成；
同音字汇词汇也应先用菜单命令数据>同音字会词汇生成；
概况应先用文本撰写，再另存为网页文件。
